## 开始使用

1. 开始使用

   在使用本框架前，请在CSS中开启border-box

   ```css
   *,*::before,*::after{
       box-sizing: border-box
   }
   ```

2. 安装salt-ui

   ```node
   npm install salt-ui-test
   ```

3. 引入
